# Apex Banking System

## Getting Started

```shell
git clone https://github.com/opeolluwa/apex-bs
```

### Dependencies

- [raylib](https://github.com/raysan5/raylib)
- [cmake](https://cmake.org/)

### Building locally

```shell
mkdir build
cd build 
cmake .. 
make 
```

### Executing program
```shell
# from the build directory
./apex_banking_system

```
